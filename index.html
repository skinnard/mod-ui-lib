Overview

This UI toolkit is a single source of truth for modzee frontend development. Consider the styles and code here to be the most current.

The following are dependencies used in frontend styling:

jquery v3.3.1 https://api.jquery.com/
bootstrap v4.4.1 https://getbootstrap.com/docs/4.0/getting-started/introduction/
popper.js v1.16.1 https://popper.js.org/
bootstrap-select v1.13.18 https://developer.snapappointments.com/bootstrap-select
Font Awesome 5 https://fontawesome.com/
